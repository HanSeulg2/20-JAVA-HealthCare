# 헬스케어 (디자인패턴)
#### 사용자 
- 관리자 : 회원가입 관리 (회원 정보 추가, 삭제, 수정), 운동 구성 관리 (추가, 삭제, 수정)
- 트레이너 : 회원 정보 조회, 운동 구성 및 회원 식단 설정, 회원에게 공지사항 알림 제공
-  회원 : 인바디 정보 (날짜, 키, 몸무게, BMI), 운동 목표 입력, 트레이너 추천 운동 구성 및 식단 조회, 운동 달성 체크
